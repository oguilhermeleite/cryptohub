# Skills Personalizadas

Esta pasta armazena skills customizadas para o projeto CryptoHub.

## Como adicionar uma skill

1. Crie um arquivo `.md` nesta pasta (ex: `deploy.md`)
2. Defina a skill no formato:

```markdown
# Nome da Skill

Descrição do que a skill faz.

## Prompt

[Instruções detalhadas para o Claude executar]
```
