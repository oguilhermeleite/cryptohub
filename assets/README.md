# Assets Folder

Esta pasta contém todos os recursos visuais do Crypto Aggregator.

## Estrutura

### `/images`
- Logo principal do site
- Imagens de banner e hero sections
- Imagens de marketing e promotional

### `/icons`
- Favicons
- Ícones de PWA (diferentes tamanhos)
- Ícones de redes sociais

## Formatos Recomendados

**Logo:**
- PNG com fundo transparente (preferencial)
- SVG para melhor escalabilidade
- Tamanhos recomendados: 500x500px (high-res), 200x200px (standard)

**Icons:**
- PNG: 16x16, 32x32, 192x192, 512x512
- SVG para ícones vetoriais

## Uso no Código

A logo principal deve ser referenciada como:
```html
<img src="/assets/images/logo.png" alt="Crypto Aggregator">
```
